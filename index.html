ABOUT


飲食系コンサルタント

飲食店の及び飲食関係の会社様向けに特化したコンサルティ
ングサービスをご提供しております。 これまで数々の飲食店
を繁盛店に変えてきたそのノウハウをご提供致します。 コン
セプト立案、店舗開発、メニュー開発、 PR 支援、従業員育
成、採用支援など店舗の出店計画から 店舗運営に関わる業務
をサポート致します。


料理やメニューの開発

食工房ではあらゆるメニューの開発を支援致します。ア
ライアンスパートナーである料理研究家の 方と連携を図
り、ジャンル問わずメニュー開発の支援が可能です。ぜひ
一度お問合せ下さい。



広告やチラシの作成・ホームページ制作

いまや飲食店を知る手段として WEB サイトの存在は欠かせ
ません。各種ポータルサイトも存在しますが、 自店のこだ
わり、コンセプト、そこで働くスタッフのことなど、自由に
表現できる場ととして、自社のサイトの 存在は非常に重要です。
お店のブランディング構築のツールとして WEB サイトを有効
活用してはかがでしょうか。 弊社では、飲食専門で WEB サイ
トの制作を行っており、商品の表現の仕方など熟知している
スタッフが多くスムーズなやり取りで 制作可能です。また、
WEB サイトだけでなく各種広告の運営やチラシやメニュー表
などの紙媒体まで制作可能です。

従業員研修

お客様と直接やり取りするのはホールスタッフになります。正
社員だけでなくアルバイトスタッフの場合もあると思いますが、
そうした重要な仕事であるにも関わらず、時間が取れない、
教育してもすぐやめるから投資する気になかなかなれない
など教育できていないという声もよく耳にします。こうした従
業員の教育を一般的な挨拶の仕方や電話の出方なども重要です
が お店のコンセプトの理解、接客の心構え、仕事の楽しさを
伝えながら、成長できる環境
